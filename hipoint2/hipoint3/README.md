# tets

Responsive HTML5 consulting website inspired by the provided reference screenshots.

## Pages
- `index.html` — home page
- `about.html` — about/story page
- `services.html` — service listing page
- `blog.html` — blog landing page
- `contact.html` — contact page
- `service-network.html` — service network page with an interactive India map
- `styles.html` — simple style guide page

## Run locally
```bash
python3 -m http.server 8000
```

Then open `http://127.0.0.1:8000/index.html` in your browser.
